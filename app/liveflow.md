## NOTE Testen in Clojurescript
- ausgehend vom lein new mies app
-- src/beispiel/main.cljs enthält code
-- test/beispiel/main_test.cljs enthält tests
- main_test muss beispiel/main und test-makros importieren
  <pre><code>
  (ns example.main-test
    (:require-macros
      [cemerick.cljs.test :refer (is deftest testing done)])
    (:require
      [cemerick.cljs.test :as t]
      [example.main :as main]))
  </pre></code>
- um in main_test nun code testen zu können, muss man folgenden code schreiben: 
  <pre><code>
  (deftest testname ; testname und funktionsname soltten gleich bzw. ähnlich sein
    (is (= erwarteter_wert (main/funktionsname arg1 ...))))
  </pre></code>
